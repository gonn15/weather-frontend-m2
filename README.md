# ClimaApp 🌤️

## Descripción

ClimaApp es una pagina web que permite consultar el clima de diferentes lugares de Chile.

---

## Temática

La aplicación está enfocada en mostrar información del clima de distintas ciudades de Chile.

---

## Modelado de datos

Los datos del clima están almacenados en JavaScript mediante un arreglo de objetos llamado `lugares`.

Cada objeto es un lugar y contiene:

- `id`: identificador del lugar.
- `nombre`: nombre de la ciudad.
- `tempActual`: temperatura actual.
- `estadoActual`: condición climática actual.
- `pronosticoSemanal`: arreglo con el pronóstico de los próximos días.

Cada día del pronóstico contiene:

- `dia`
- `min`
- `max`
- `estado`
